# 0.1.0 Release-Candidate Rehearsal

Date: March 12, 2026
Status: local rehearsal complete, not published

## Release Scope

This rehearsal covers the first crates.io release candidate for `jepa-rs`
without publishing any crates.

Included scope:

- alpha Rust JEPA workspace on top of `burn` 0.20.1
- strict image and video masked forward helpers
- strict I-JEPA image differential parity across three checked-in fixtures
- safetensors loading plus ONNX metadata inspection and initializer loading

Explicit limits:

- `JepaComponents::forward_step` remains approximate and is not the strict semantic reference
- differential parity is image-only for this release candidate
- ONNX runtime execution stays out of scope
- the project is not called production-grade yet

Suggested known-limitations section:

- Differential parity proves only strict image flows today.
